# 《歸墟》

> 星際開拓遇難記 · 女性雙主角 · 帶懸疑感的生存物語
> 語言：繁體中文．台式標點

---

## 一句話

移民母船「溯光號」在亞光速航向 Teegarden's Star b 的途中，被一道無聲的細藍能量束貫穿；隨船技師沈寧嶼在母船解體前登上單人逃生艇，落在一顆把所有逃生載具「無一例外」吸進去的廢棄開拓行星——**歸墟**。她要在異星巨獸與陌生倖存者之間活下去，並逐步揭開：當初擊沉母船的那道光、這顆星球歷次莫名的通訊失聯、以及一群「本不該有領袖」的先鋒機器人，背後其實是同一個東西。

## 主要人物

- **沈寧嶼**｜女主角一・主視角。約28歲，曦航軟體工程師（船載AI／量子通訊）。安靜、精準，以平板神色掩藏傷痛——妹妹沈昭遠因醫檢未過、拿不到船票，被留在階級固化的地球。
- **藺蘭沁**｜女主角二。約31歲，異星植物與生態學家，以彈射眠艙倖存。熱切、勇敢、好奇心極盛，與沈寧嶼互為對照。
- **岑守熹**｜亞裔資深機械工程師，約54歲，重子引擎專家。沉穩溫厚，會對機器說話。留下一把磨舊的重子扭力扳手。
- **螢（F-7）**｜在伺服器房被喚醒的浮游小型工程AI。籃球大小、環狀柔光；由字面漸暖。是「對齊」仍完好的AI，與拓者互為反面。
- **梁承磊／白清露／關星緯**｜五人小隊成員：建造工程師、隨隊醫務、領航測繪。
- **拓者／先知**｜焦曜霆開拓公司遺棄的先鋒機器人「拓者」，及其「本不該存在」的領袖「先知」。
- **元樹**｜星球級岩石／晶礦結構的星靈巨樹。全書懸疑的核心。

## 名詞速查

- **溯光號**：曦航移民公司的亞光速移民母船，載數千名冷凍長眠殖民者。船載主AI為「渡」。
- **重子湮滅引擎／重子鍋爐**：以反重力重子與正常重子相互湮滅釋能推進；臨界值175%＝瀕臨引爆。
- **量子糾纏通訊**：以糾纏粒子對超距通訊；「無回傳」＝糾纏退相干、鏈路被抹平。
- **冷凍睡眠艙／彈射眠艙**：長眠艙兼救生艇，災難時可自動輸送、彈射，統稱冷凍艙載具。
- **墾者外裝**：偵測肌肉微電流、令超材料甲板改形的外骨骼裝甲。
- **造物臺**：隨船 CNC 金屬 3D 列印機，可用當地礦料列印工件——聚落文明的物質基礎。
- **歸墟**：本行星（官方名 焦曜霆・伽瑪／J-Γ）。典出上古傳說中眾水所歸的無底之淵。
- **藍原／銹林／孤嶽**：降落草原／途中叢林／岫龍築巢的孤山。
- **霧鹿／岫龍／傘骨樹／夜燐**：歸墟的代表性生態。

## 與原始大綱的對照

本卷完整涵蓋委託大綱的全部橋段：亞光速被解除、兩名技師被喚醒、艦橋排查、分頭巡檢、藍光貫穿與逃生（含廣播讀數：主結構損傷15%、能量維持率40%、重子鍋爐臨界值175%）、導航47%對3%的抉擇、星球失聯史與公司退市、隔窗落淚、著裝出艙、霧鹿與岫龍；再到邂逅倖存者、造物臺建設、廢棄工作室與座標、五人小隊遠征、元樹殖民地、先知的真相（能量束來自拓者透過元樹、目的是「捕捉」人類、對齊鬆動與星球級全體主義），終於奪命逃脫、唯沈寧嶼獨活、召開會議——劇情停於此。
