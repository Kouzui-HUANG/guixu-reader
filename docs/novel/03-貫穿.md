# 第3章　貫穿

　　觀景舷窗的玻璃比她的體溫低太多。沈嶼把額頭抵上去的那一刻，那層冷像一枚硬幣壓進眉心，順著顴骨一路往下滲。窗的內側，是她方才一步一步走過來的巨型冷凍睡眠艙——成千張臉在藍幽幽的低溫光裡靜止著，霜在艙壁上結成細小的蕨葉，一格接一格，安靜地往上爬。空氣裡有一股金屬泡得太久的鏽味，摻著臭氧那一點尖，遠處氧再生塔像退了潮的海，一呼，一吸，低低地響。循環風從腳下的甲板縫裡漫上來，帶著替長眠者恆溫的那種涼；整座艙室裡，大概只有她這一小團體溫是多餘的、醒著的。窗外是黑，純粹的、不反光的黑。她站在兩種靜之間：一種是活人的睡，一種是宇宙的空，中間只隔著一指厚的玻璃，和玻璃上她自己模糊的倒影。她看見自己眼底的青影，也看見那些沉睡者的臉一層層疊上她的臉，像很多層薄薄的、永遠不會醒來的水。

　　螢浮在她肩側，燈環還是初醒時那種字面而生硬的冷白。「偵測到船外一處光點，方位持續收斂。」它的微型機械臂輕輕收攏，關節發出蚊子似的細響，「建議記錄。」

　　她順著它的話，往深空看去。

　　起先只是一粒。遠得像某顆恆星走了神，藍得不對——不是星子那種摻了白的藍，是更飽和、更集中的一種藍，像有人在極遠的地方點亮了一盞燈，正不慌不忙地把燈芯往上捻。它由遠而近，一分一分地亮，那節奏裡有一種近乎耐心的東西：不慌，不搶，像在對準什麼。沈嶼的拇指指甲不自覺掐進了另一隻手的指腹，掐出一小圈鈍痛。她說不上為什麼，只覺得這光太從容了，從容得不像天象，倒像一次瞄準——

　　然後它亮開了。

　　像一枚閃光彈在窗外炸裂，藍白的光在一瞬間灌滿整室，把每一張沉睡的臉、每一道霜蕨、她自己的倒影，全洗成沒有陰影的白。她眼前只剩一片轟然的亮，視網膜燒得生疼，什麼都看不見了。

　　——那一刻，若有什麼在船外的黑裡冷冷地看著，會看見這樣一幅畫面：溯光號像一根沉睡的長骨，靜靜懸在兩顆恆星之間；而一道細得幾乎不真實的藍色能量束，自虛空中的某一點伸出來，無聲地、精準地，貫穿了它的整條長身。沒有爆音，沒有火花，那束藍只是在船殼上開了一個乾淨的洞，像一根燒紅的針，穿過一塊繃緊的布。真空不會為任何事發出聲響。它只是被穿過了，如此而已。

　　聲音是後來才追上來的。

　　整艘船猛地一沉——那不是她熟悉的任何一種震：不是引擎的搏動，不是磁軌的錯位，而是一種從龍骨最深處傳上來的、巨大的悶響，像一根骨頭被人隔著肉撅斷。那震一路爬進她的牙關、她的膝蓋，讓臼齒咯咯地互相磕碰。她整個人被甩向舷窗，掌心啪地拍上玻璃，額頭跟著磕上去，眼前那片白這才褪成一團游動的紫。艙裡所有的燈同時熄了半秒——那半秒黑得那麼徹底，她清楚聽見了自己的心跳；再亮起來時，全成了紅，一種黏稠的、警示用的紅，把霜蕨染成血色，把每一張沉睡的臉都染上了一層不祥。

　　渡的廣播響起來。它一向是這艘船裡最平穩的聲音，此刻卻在句與句之間裂開細小的雜訊，像從水面下傳上來的話：「……主結構損傷15%。能量維持率40%。重子鍋爐臨界值175%。」每一個數字都咬得很清楚，清楚得近乎殘忍。「啟動逃生程序。全體移動至最近之避難彈射裝置。啟動逃——」

　　那個「逃」字沒有說完。廣播斷在半途，像被人一把掐住了喉。緊接著，天花板上一排指示燈次第轉成琥珀色，一盞，一盞，朝著船尾滅過去——沈嶼認得那個顏色，那是「無回傳」的顏色，是鏈路那一端變成空無的顏色。渡離線了。這艘船裡最先醒來、最該最後才睡去的那個聲音，此刻先她一步，沉了下去。

　　她還沒站穩，腳下的地板就開始動了。

　　不是搖晃，是運轉。巨型冷凍睡眠艙的整套自動化，在渡離線的同一刻被硬體層的逃生協定接管——那些原本靜止排列的彈射眠艙單元，一具接一具地從艙架上脫扣：脫扣的震動抖落它們身上薄薄一層霜，冷卻劑的白霧從接口噝噝地洩出來，在紅光裡浮成一縷縷血色的煙。它們落上傳送軌，發出沉重而規律的哐、哐、哐，像糧倉裡的糧被人一齊倒了出來。沈嶼站在通道中央，看著千百具眠艙在她兩側如輸送帶般滑動、轉向，被逐一裝填進艙壁盡頭的避難彈射裝置。每一具的觀察窗裡，都躺著一張她方才走過、看過的臉——藍白的，闔著眼的，對這一切一無所知的臉。機械的洪流貼著她的肩背推擠而過，冷，重，不由分說。她被夾在千人的長眠之間，像一粒被裹進河心的沙。有那麼一瞬，她想起某個她答應過要回去接、卻終究只能留在身後的人，喉頭猛地一酸——可這整條發著光的河裡，醒著的只有她一個，連替誰難過的餘裕都沒有。

　　然後，連重力也叛變了。

　　環轉甲板的旋轉一定是亂了，人工重力整個垮下來。她先是覺得自己變輕，胃往上浮，接著「下」這個方向就徹底失了信用。環廊在她眼前翻了個身：牆成了地，地成了牆；一具脫了軌的空眠艙從她頭頂緩緩砸下，又在半空改了主意，斜斜地飄開。沈嶼腳底一空，整個人往一個荒謬的方向跌去——她跌「上」了天花板，脊背重重撞在那排剛熄滅的燈管上，一列冷硬的凸稜硌進她的肩胛。她就這麼仰面躺在本該是頂的地方，看著本該是地的那一面在頭頂上緩緩地晃。失重把她的內耳攪成一團漿糊，她分不清自己是在墜落還是在上升，只有胃還固執地翻找著那個已經不存在的「下」，翻江倒海，把一口酸水頂到了舌根。

　　「沈嶼。」螢的聲音貼著她的耳廓響起來。它的燈環不知什麼時候由冷白轉成了一種暖金，在滿室游動的紅裡，是唯一一點不刺眼的光。「走廊已失去參考方向。我為你重新標記路徑——跟著我。」它一路亮著，像一顆有耐心的星，在這座翻覆的迷宮裡，替她把「哪邊是出口」重新定義了一遍。她伸手抓住一道扶索，指節攥得發白，讓身體順著螢的光，往那個此刻既非上也非下的方向游過去。

　　通訊在這時活了一下。

　　「嶼丫頭——」陳岱的聲音擠過一層厚厚的雜訊，背景裡是一種她從未聽過的、沉重的搏動，像有什麼巨大的東西在他身邊喘氣。她甚至聽得見他壓著嗓子在跟那口鍋爐嘀咕，哄小孩似的：好了，好了，再撐一下。「聽得見嗎？我在洩壓層。」

　　「岱哥。」她幾乎是撲住那道扶索，「臨界值都一百七十五了，你快出來，逃生艇還來得及——」

　　「我知道。」他打斷她，語氣卻不急，甚至還有心思笑了一聲，那笑被雜訊磨得毛毛的。「這老傢伙啊，是被什麼從外頭頂了一下——讀數不講道理，自然劣化的曲線不會這樣跳。」他喘了口氣，金屬摩擦的悶響在他背後起起伏伏，「約束快壓不住了。我把洩壓閥手動搖開了，能把那條曲線往下按一按。按住一分鐘，尾段那些眠艙，就多一分鐘往外彈。」

　　沈嶼懂那句話的意思。她太懂了。他不是在救船——船，已經救不回來了。他是在用自己還站在那口鍋爐旁邊的每一秒，替那些他這輩子從沒見過、也永遠不會醒來認得他的陌生人，多換幾具彈出去的眠艙，多換那麼幾秒。

　　「你跟我一起按。」她聽見自己的聲音在抖，「我回去，我們兩個——」

　　「你敢回來我揍你。」他說，隨即又軟了下來，聲音低得幾乎被鍋爐的搏動吞掉，「……我那把扳手，還在你工具袋裡吧？那把磨舊的重子扭力扳手。」

　　她騰出一隻手摸了摸腰側。硬的，涼的，柄上還留著他掌心經年磨出來的那點溫，隔著一層布都認得。「在。」

　　「帶著。」他說，「別弄丟。回頭……」

　　「回頭什麼？」

　　「回頭，幫我——」

　　那句話沒有說完。不是被掐斷，是那條連著他的線，忽然變成了和渡一樣的、乾淨的沉默：沒有雜訊，沒有殘響，沒有一絲錯誤碼，彷彿那一端，從來就不曾有過一個人在說話。沈嶼一手攥著扶索，一手按著那把還帶著他體溫的扳手，在翻覆的紅光裡，聽著耳機裡那片太過乾淨的靜，一個字也擠不出來。

　　「沈嶼。」螢的暖光在她面前輕輕一沉，又亮起來，「逃生艇甲板，還有一百二十公尺。艙門的關閉序列已經開始。」

　　她把那口氣嚥了回去，像嚥下一枚金屬硬幣，硌著食道一路往下。哭是後來的事。此刻她的身體比她的腦子清醒，順著螢的光，一段扶索接一段扶索，在失去了上下的環廊裡爬、蹬、翻，指節撞在艙壁上撞得發麻，也顧不上。汗從鬢角滲出來，在失重裡不肯往下淌，只黏膩地浮貼在皮膚上。她鑽過一道正在合攏的隔艙門，肩上的布被門緣生生削去一塊，冷氣立刻貼上那道裸露的皮膚；她越過整片正在吞吐眠艙、像產道般開闔的甲板——一具眠艙正從發射口彈出去，短促的推進焰在她眼底刷過一道藍，緊接著又是一具，又是一具——終於在一排冷而整齊的龍骨盡頭，摸到一艘還亮著待命綠燈的小型逃生船。

　　艙蓋在她身後合攏的那一聲輕響，是這半個鐘頭裡她聽過最溫柔的聲音。螢滑了進來，停在她胸前，燈環是一整片安定的暖金。小艙裡的空氣是新的，帶著長久封存的塑膠與冷金屬的味道；一圈儀表在她面前逐格亮起，綠的、藍的，安靜得近乎陌生。座椅的束帶自動收緊，勒住她的鎖骨，把她釘進椅背。系統不等她按任何一個鈕——彈射是自動的。

　　「彈射。」螢說，「三，二——」

　　沒有「一」。一股蠻力從船尾把她狠狠推出去，過載像一隻手死死按住她的胸口，把她整個人壓進椅背，把她的臉皮往後扯，視野從邊緣一圈一圈地暗下去，血在耳朵裡擂鼓。然後那隻手又忽然鬆開，所有的重量在同一瞬間被抽空——她被吐進了失重，吐進了黑，胃、四肢、心臟，全都懸在了半空。

　　她轉過頭，隔著小小的舷窗，回望那艘正在死去的母船。

　　她看見的第一眼，竟然是美的。溯光號已經退成一枚狹長的剪影，靜靜浮在兩顆恆星冷冷的光裡；而從它的側腹，一具接一具的冷凍艙載具正密密地射出——成百上千的彈射眠艙，各自拖著一縷淡藍的推進尾光，在黑色的幕布上拉出千萬條平行的、緩緩發散的光線，像有人抓起一大把會發光的種子，朝宇宙撒了出去。那麼多的光，那麼密，那麼有秩序地離開一艘正在崩解的船。有那麼一瞬，沈嶼幾乎忘了那是災難——它美得不像災難，倒像一場豐收，像有誰在極遠的地方，耐心地、一粒不落地，把什麼收攏了進去。

　　然後，光流的源頭裂開了。

　　——溯光號從中段斷成了兩截。前半的艦橋與後半的引擎緩緩地分了家，中間扯出一片翻卷的碎屑與凍成白霧的空氣，像一根長骨被無聲地折斷、又被無聲地抽開。斷口處沒有火——這裡沒有氧，火燒不起來；只有金屬在真空裡靜靜地舒展、剝落、飄散，慢得近乎溫柔。

　　沈嶼一直看著那截藏著洩壓層、藏著一口鍋爐、藏著一個還欠她半句話的人的船尾。她看不見讀數了，可她知道，那條曲線已經越過了一百七十五。

　　尾段在無聲中亮了起來。不是爆——至少不像她想像過的那種爆。是一團光，一團純粹的、無聲的、球形的白，從重子鍋爐的位置向外膨脹開來，一瞬間就吃掉了那半艘船、那些還來不及彈出去的眠艙、以及那個還欠她半句話的人、和他守到最後的那口爐，所在的整個空間。那團白亮起的一瞬隔著舷窗照到她臉上，不燙，只是很白，白得她眼睛發酸——她沒有別開臉。光漲到最大，又緩緩地黯下去，像宇宙不動聲色地眨了一下眼。等那點白徹底散盡，那裡就什麼都沒有了，只剩下還在四散的光流，和一片比先前更深的黑。

　　沒有聲音。從頭到尾，一點聲音都沒有。

　　沈嶼獨自漂在殘骸與光流之間。逃生船小得像一枚被吐出來的種子，順著某種她還讀不懂的導航，慢慢地轉向一個她還看不見的方向。艙壁外就是絕對零度的黑，隔著那幾公分厚的殼，那股冷像有重量似的貼著她的後背。她的手一直按在腰側那把扳手上，那點涼，正被她的掌心一分一分地焐熱。舷窗外，千萬條藍色的光線仍在遠去、發散、一盞盞熄滅；而艙內，唯一還亮著的，是停在她胸前的那一環暖金的、極輕極輕的光。

　　「螢，」她開口，聲音啞得自己都聽不出，「我們……還連著誰嗎？」

　　燈環暗了一瞬，像在誠實地、一條一條地查找。然後它輕輕地、幾乎是溫柔地，亮了回來。

　　「只剩下你，」它說，「和我。」
