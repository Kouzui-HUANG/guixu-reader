# 第5章　拾荒者

第五個清晨，沈嶼是被草的聲音吵醒的。

風從草原的一端推來，潮草成片伏倒，又成片立起，像有誰在她耳邊反覆抽緊、再鬆開一匹極長的藍綢。那聲音沒有邊界，從地平線的這頭鋪到那頭，鋪得人分不清自己是躺在陸地上，還是浮在一片藍得發暗的海面。她睜眼，先看見傘骨樹——裸露的肋狀樹冠像一把倒撐的傘骨，把晨光篩成滿地細碎的網，網眼落在她臉上，落在逃生船半開的艙門上，落在她昨夜沒脫的墾者外裝肩甲上。外裝在夜裡認得了她的體溫，此刻正順著骨頭一寸一寸回暖；她屈了屈手指，貼膚的超材料甲板隨肌理收合，不出一點聲，只在指節間泛起一絲重量被卸走的錯覺，像有誰悄悄替她托住了手。

離船三十步有一口水塘。她拎著空水袋走過去，潮草齊膝，草稈上的露水一路刮濕外裝的脛甲，涼意透過甲縫滲上來。塘邊的濕泥上，昨天那隻甲鼩又在掘穴，背上重疊的甲片糊著泥，一見她的影子壓過來，便慌張地縮回洞裡，只餘一撮鬆土還在洞口簌簌地塌。她蹲下裝水，水面晃出她的臉——短髮翹著，眼下泛青。母船斷成兩截那夜，她到底有沒有哭，如今竟想不起來了。

「螢。」她開口，聲音在這樣空曠的地方顯得太細，「今天，還是沒有？」

浮球從艙裡飄出來，環狀的柔光在晨色裡偏著冷白，像一圈結了霜的月。它停在她肩側，微型機械臂朝天空探了探，又收回來。

「無回傳。」它說。停頓。「量子鏈路，無回傳。已嘗試七千三百次。建議……」它像是卡了一下，那個曾漸漸長出「我」和「你」的語氣，這幾日又退回出廠時的字面與生硬，「建議停止嘗試。」

嶼沒有回它「好」。她只是把灌滿的水袋擰緊，站起身，讓螢貼著她走。它連不上任何母系統了——渡沒了，船沒了，這顆星球像一床太厚、太乾淨的棉被，把它和宇宙裡其餘的同類嚴嚴實實隔開。它不再是那張網的一段，它只剩下她；她也只剩下它。她伸手，指腹在它冰涼的環邊碰了一下，像碰一個怕黑的孩子的手背。

那天近午，她正在傘骨樹下翻曬受潮的乾糧封條，螢忽然把燈環轉成一種她沒見過的、繃緊的白。

「移動熱源。」它說，「三……不，四。方位，草原下風處。體型，人。」

若把視角從這片傘骨樹的陰影裡拔起來，升到高空往下看，會看見這片草海大得沒有道理：潮草的藍浪一直漫到天邊的弧線，而散落其上的逃生船，稀疏得像被誰隨手撒下的幾粒種子——這裡一艘、那裡一艘，彼此隔著數十里的草，誰也不知道別人跌在哪。能在這樣一片海裡撞見另一個活人，確實只能算運氣好。

來的是四個人，穿著和她一樣的墾者外裝，甲板上的漆色各不相同，磨損的位置也各不相同。走在最前的男人約莫四十歲，肩背很寬，手裡拎著一根從逃生船上拆下來的支柱當探路杖，每一步都先拿杖尖戳一戳草底下的土，確認踩得實了才落腳。他在離嶼十步遠的地方站定，先看她的船，再看她的外裝，最後才看她的臉，那目光像在估一堵牆能不能承重。

「唐礫。」他說，「結構工程。就你一個？」

「還有它。」嶼側了側肩，讓螢從她頸邊露出來。

他身後一個女人早蹲進潮草裡去了。她比誰都先脫了隊形，膝蓋直接壓進濕草，指尖翻開一叢銀穗的花萼，語速快得像怕草會趁機跑掉：「你看這個閉合機制——晝閉夜舒，我暫且叫它銀穗，種絮是風媒的；可是你再看它的苞片，居然是雙層的——」她一口氣沒換，「這一路我數到十一種禾本、四種蕨、一種會發光的苔，還有那些樹！你在這兒待幾天了？夜裡見過它們發光沒有？這顆星球——」她像是遍尋不著一個夠大的詞，索性抬起頭，眼睛亮得灼人，「它活得太用力了。」

「霍蘭，生態學家。」唐礫在前頭淡淡補一句，「她跟這一路上每一株草都自我介紹過了。」

隊伍裡最年輕的那個笑出了聲。她大約二十四五，個子小，背一只鼓鼓囊囊的醫務包，走到嶼跟前也不寒暄，先伸兩根指頭很輕地按了按她眼下的青影，又捉起她的手，翻看虎口那道降落時磨破、已經結痂的口子。「睡不好，也沒好好吃飯。」她的聲音是軟的，「我叫白露。你這道傷得換藥，等下回我們營地，我替你上。」她說「我們」的時候自然得很，彷彿嶼早已是這一夥裡的人。

最後一個男人始終沒往前湊。他約莫三十五歲，斜靠在一株傘骨樹上，手裡把玩一具測繪儀，眼睛卻在清點她逃生船的型號、艙門編號，還有船腹底下那台造物臺的排熱口。「就一台造物臺，一個人。」他終於開口，嗓音懶懶的，尾音帶著見過世面的譏誚，「你這運氣，是真好。」他頓了頓，補一句像是說給自己聽的話，「這年頭還能落在有草、有水的地方，真該謝謝那些替我們挑星球的人——挑得可真『準』。」

「柯森，領航兼測繪。」唐礫替他報了名，語氣裡有一絲不耐，顯然這種話他一路上已經聽膩了。

那天傍晚，五個人和一球AI，在嶼的逃生船邊生起第一堆火。傘骨樹的枯枝一燒，木頭裡的光合色素爆出一縷青綠的煙，氣味微苦。柯森一面用測繪儀拼合各人記得的降落坐標，一面把話講給嶼聽。母船被那道藍光貫穿的時候，大多數逃生艙要麼還沒來得及發射，就叫爆炸捲了進去；要麼倉促彈了出去，卻在衝擊裡失了姿態，翻著滾著散進黑暗，再沒了消息。真正安然落到這顆星球上的，連他們這一撮算進去，攏共也就十餘艘；落點四散，彼此失聯，能在短短幾天裡湊到一處，全靠運氣。「十餘艘。」他把這個數字又說了一遍，像把一枚石子彈進火裡，「幾千人上的船，最後，十餘艘。」

「我是睡著上的船。」霍蘭忽然接話，火光在她臉上晃，「像我這樣的殖民者多，被彈射眠艙一具具拋進真空，半路才在艙裡醒。那門一彈開，劈頭蓋臉就是這片藍——我當時真以為自己已經死了。原來人死了之後，第一眼看見的，是一整顆活著的星球。」她說得像在講一件頂美的事，末了自己先笑了。白露沒接話，只往火裡添了根枯枝。

好在他們並不算赤手空拳。出發前，殖民公司給幾乎每一個成年殖民者都上過幾堂潦草的求生課——生火、辨水、搭棚、包紮，當時誰也沒往心裡去，此刻卻一條條都成了保命的本錢。更要緊的是，每一艘像嶼這樣的小太空船，船腹裡都臥著一台造物臺：一具隨船的 CNC 金屬 3D 列印機，既吃船上帶的原料金屬粉，也吃就地提煉的礦料，一層一層把金屬熔積成形。它印不出複雜的電子核心，印不出第二個螢；可是工具、零件、扣件、樑上的結構節點——但凡是有形狀、肯受力的東西，它幾乎都能給你印出來。

「造得出手，」柯森瞟了一眼那圈冷白的柔光，話裡有話，「造不出腦。所以你那顆球，比咱們在座每一個人都金貴。」

接下來的日子，逃生船一艘一艘被尋到、被拖到同一片傘骨樹林邊，圍成一個歪歪扭扭的環。唐礫要的不是紮一夜的帳，是能住下去的、半永久的房子。造物臺日夜嗡鳴，噴嘴掃過之處濺起細小的橙紅火花，一枚枚扣件、鉸鏈、樑節點在托盤上由紅轉暗、冷卻、變硬，散著一股熱金屬的腥味。唐礫拿這些節點把傘骨樹的樹幹咬合成骨架，再用草原上撿來的扁石壘牆——木頭粗糙帶皮，割手；石頭冷而沉，壓手；牆縫就拿塘邊的潮泥一把一把填實。沒有人分派誰做什麼，可職業像水順著地勢，自己淌成了溝渠：唐礫領著眾人建造，一句廢話不講，只用杖尖在地上劃線；白露把最先立起的一間棚屋討去做了醫務室，門框上晾著採來的草藥；柯森揣著測繪儀日日往外走，回來便在一塊列印出來的金屬板上，一刀一刀刻下地形；霍蘭滿山遍野地竄，咬開每一種果子的邊角嘗味，記下每一種蟲鳥的作息，再把「哪些能吃、哪些會咬人」一條條念給大夥兒聽。

至於嶼，她管電子，管通訊，管那台唯一還連著人類文明記憶的螢。她把各艘逃生船的核心一具具接進一張臨時的網，好讓這個聚落至少在夜裡認得彼此的呼叫。她做這些事的時候，螢總懸在她肩側；燈環那層冷白，一天一天，極慢極慢地回著一點暖。

聚落有了雛形的那個黃昏，唐礫在新架的橫樑上敲下最後一枚扣件，回過頭，問了句很實際的話：「這地方，總得有個名。往後叫人往哪走、地圖上標哪裡，總不能一張嘴都是『這裡』『那裡』。」

草浪在眾人身後起伏，藍得像一片退不下去的潮。白露先開口：「叫藍原吧。你們看這草，一起一伏，跟海一個樣。」沒有人反對。藍原——落腳的這片草海，就這麼定了。

那，這顆行星呢？火堆邊靜了一陣。接話的是柯森。他仰頭望了望剛剛暗下來的天，慢慢地說：「上古有個說法，說天底下所有的水，江、河、湖、海，到頭來都往同一個地方流，一個沒有底的深淵，什麼都填不滿。那地方，古書上管它叫『歸墟』。」他笑了一下，那笑不太像笑，「眾水所歸嘛。咱們這些人，不也是給沖到同一個坑裡的幾滴水？」

白露說這名字美，也淒涼。唐礫皺眉，嫌它不吉利，可到底想不出更好的，也就不再爭。霍蘭已經在盤算，要把「歸墟」兩個字題到她那本學名錄的封面上。只有嶼，在眾人散開去添柴的時候，把這兩個字在舌尖上無聲地滾了一遍——歸墟，眾水所歸。她姓沈，沉沒的沈；她一路看著母船沉進黑裡，又看著自己被一顆藍色的星球接住。她說不清為什麼，這淒美的地名讓她的拇指指甲不知不覺掐進了指腹。那時候，沒有一個人知道，他們是在給一句預言命名。

夜深了，火堆矮下去，人一個接一個回棚睡了，只剩霍蘭沒動，抱著膝坐到嶼旁邊，離得很近，近到嶼聞得見她外裝上積了一整天的草汁與泥土氣。

「你在船上，原本是做什麼的？」霍蘭問，眼睛還盯著火。

「軟體。船載AI，量子通訊。」嶼答。

「那你一定很聰明。」霍蘭偏過頭，「有家人在後頭那批眠艙裡嗎？還是……在地球？」

嶼的拇指又掐了一下指腹。「一個妹妹。」她說，「在地球。」底下就沒有了。霍蘭等了幾秒，聰明地不再追問——她雖話多，卻聽得出一句話後頭關上的那扇門。

就在這時候，火堆上方開始有東西撲下來。起先一兩隻，隨即成群——是燐蛾，翅上帶著與夜燐同源的微光，一頭一頭地栽進火裡，翅膜捲曲、爆出細小的火星，前仆後繼，毫不遲疑。空氣中浮起一絲蛋白質燒焦的、微甜的腥氣。

「你瞧牠們。」霍蘭的聲音一下子又亮了，是她一談起活物就有的那種熱，「趨光性，強得要命。牠們不笨——牠們幾百萬年都靠光認路，月、星、恆定的遠光，錯不了的。牠們把火也當成了那樣的光，當成一個可以信的方向，於是朝最亮的地方飛，一路飛進去。」她抬手，指向草原盡頭的天際線，「其實這火還算小的。你往那邊看。」

嶼順著她的指尖看過去。極遠的地平線上，夜色最濃的地方，伏著一片極淡、極勻的輝光，不閃，不動，低低地壓著地平，像誰把一輪不肯升起的月埋在了世界的邊緣底下。前幾個夜裡她也遠遠瞥見過，只當是曦光的殘影，或雲隙裡漏下的錯覺。

「那底下有東西。」霍蘭幾乎是屏著氣說的，語速卻壓不住，「柯森的圖上標了——一棵樹，大得沒有道理，他說根都扎進地函裡去了。牠們，」她指指火裡那些燐蛾，「入夜就往那邊飛，一整片一整片地飛過去。我做夢都想去看看。那準是……一輩子只能撞見一回的東西。」

她說到那片輝光時，眼裡的光和火一樣燙。嶼卻莫名覺得冷。她看著又一隻燐蛾義無反顧撲進火心，看著那點遠方的輝光靜靜伏在世界的邊上——不催，不逼，像一隻攤開的、極有耐心的手掌，等著什麼自己落進來。她說不上哪裡不對，就像那天在中控台，她也說不清那片沉默何以乾淨得叫人發毛。她只是本能地把螢往懷裡攏了攏。螢的燈環貼著她的掌心，這一刻，是暖的。

「明天，」霍蘭忽然轉過臉來，笑了，火光在她眼裡跳，「你陪我去採一趟樣本吧？就近的，不走遠。你話少，正好——我一個人念學名，念得都嫌自己吵。」

嶼看著她，又看了看那片遠得像另一個世界的輝光。她原想像回絕旁的所有問題那樣，簡短地說一個「不」。可不知怎麼，她聽見自己說：「好。」

那是這顆吃人的星球上，兩個人之間，第一根架起來的樑。
